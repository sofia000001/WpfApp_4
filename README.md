### Задача 22. Серия и номер российского паспорта

**Описание**: Паспорт гражданина РФ имеет серию из 4 цифр и номер из 6 цифр, разделённые пробелом или дефисом. Серия не может начинаться с "00", номер не может быть "000000".

**Регулярное выражение**:
```regex
\b(?!00)\d{4}[ -](?!000000)\d{6}\b


**Пояснение обозначений**:

| Обозначение | Значение |
|-------------|----------|
| `\b` | Граница слова (целое число) |
| `(?!00)` | Негативный просмотр вперёд: первые две цифры не могут быть 00 |
| `\d{4}` | Ровно 4 цифры (серия паспорта) |
| `[ -]` | Допустимый разделитель: пробел или дефис |
| `(?!000000)` | Негативный просмотр вперёд: номер не может состоять из шести нулей |
| `\d{6}` | Ровно 6 цифр (номер паспорта) |
| `\b` | Граница слова |

### Примеры корректных строк:
- 4512 345678
- 1203-901234

### Примеры некорректных строк:
- 4512 000000 (номер состоит из нулей)
- 4512345678 (отсутствует разделитель)
- 123 456789 (не 4 цифры в серии)

### Задача 14. Инициалы и фамилия (И.О. Фамилия)

**Описание**: Формат: заглавная буква имени + точка + заглавная буква отчества + точка + пробел + фамилия. Фамилия начинается с заглавной буквы, остальные строчные. Допускается дефис в фамилии.

**Регулярное выражение**:
```regex
\b[А-ЯЁ]\.([А-ЯЁ]\.)\s[А-ЯЁ][а-яё]+(-[А-ЯЁ][а-яё]+)?\b

**Пояснение обозначений**:

| Обозначение | Значение |
|-------------|----------|
| `\b` | Граница слова |
| `[А-ЯЁ]\.` | Заглавная русская буква и точка (имя) |
| `([А-ЯЁ]\.)` | Заглавная русская буква и точка (отчество) |
| `\s` | Обязательный пробел |
| `[А-ЯЁ][а-яё]+` | Фамилия: первая заглавная буква, остальные строчные |
| `(-[А-ЯЁ][а-яё]+)?` | Опциональная дефисная часть для двойных фамилий |
| `\b` | Граница слова |

### Примеры корректных строк:
- И.И. Иванов
- А.С. Пушкин

### Примеры некорректных строк:
- И. Иванов (нет отчества)
- И.И. иванов (фамилия с маленькой буквы)
- И.И.Иванов (нет пробела между инициалами и фамилией)

### Задача 6. Дата MM/DD/YYYY с учётом високосных лет

**Описание**: Дата в формате месяц/день/год. Месяц: от 01 до 12, день — с учётом реального количества дней в месяце, включая 29 февраля для високосных годов.

**Регулярное выражение**:
```regex
\b(0[1-9]|1[0-2])\/(0[1-9]|[12][0-9]|3[01])\/(19|20)\d{2}\b

**Пояснение обозначений**:

| Обозначение | Значение |
|-------------|----------|
| `\b` | Месяц от 01 до 12 |
| `\/` | Разделитель (слеш) |
| `(0[1-9]|[12][0-9]|3[01])` | День от 01 до 31 |
| `\/(19|20)\d{2}` | Год: 19xx или 20xx (диапазон 1900–2099) |
| `\b` | Граница слова |


### Логика проверки високосности:

- if (год % 400 == 0) → високосный
- else if (год % 100 == 0) → не високосный
- else if (год % 4 == 0) → високосный
- else → не високосный

### Примеры корректных дат:
- 02/29/2020 (високосный год)
- 12/31/1999
- 01/15/2024
- 02/28/2021

### Примеры некорректных дат:
- 02/29/2021 (не високосный год)
- 04/31/2023 (в апреле 30 дней)
- 13/01/2023 (неверный месяц)
- 02/29/1900 (1900 не високосный)
